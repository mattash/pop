Pop!

enjoy simple dropdown menus with pop!, an unobtrusive javascript plugin for jquery.

Demo

Download

Tar or Source or Github

Usage

Include the stylesheet, with any of your mods.

<link href="/pop/stylesheets/pop.css" media="all" rel="stylesheet" type="text/css"/>

Include jQuery and jquery.pop.js

<script language="javascript" src="/pop/javascripts/jquery.js" type="text/javascript"></script>
<script language="javascript" src="/pop/javascripts/jquery.pop.js" type="text/javascript"></script>

Initialize pop!

<script type='text/javascript'>
   $(document).ready(function(){
     $('.pop').pop();
   });
</script>

Wrap your popped element with your desired pop class.

<div class='pop'>
   <p>you can put anything you want in here!</p>
   <p>images, links, movies of your cats. you name it!</p>
</div>

Options

There are a few options you can choose from to suit your needs

toggle_text: string (empty by default)
The text which is displayed on the toggle div.
toggle_class: string (".pop_toggle" by default)
The class that is applied to the toggle div.
wrapper_class: string (".pop_wrapper" by default)
The class that is applied to the wrapper of the dropdown menu.
hang_left: true or false (false by default)
Adds class to menu that will force it to hang left. This is done automatically if the menu is found to render outside of the viewport.
hang_under: true or false (false by default)
Adds class to menu that will force it to hang under the toggler.
initial_zindex: number (1000 by default)
The minimum z-index applied to the wrapper element.
